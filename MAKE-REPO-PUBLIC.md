# Make Repository Public for GitHub Pages

## Quick Steps

1. Go to: https://github.com/kowalcg/ChocolateOnJames/settings
2. Scroll down to the **"Danger Zone"** section at the bottom
3. Click **"Change visibility"**
4. Select **"Make public"**
5. Type the repository name to confirm: `kowalcg/ChocolateOnJames`
6. Click **"I understand, change repository visibility"**

## After Making Public

1. Go back to: https://github.com/kowalcg/ChocolateOnJames/settings/pages
2. You'll now see the "Source" options
3. Select:
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)
4. Click **Save**

Your dashboard will be live at: **https://kowalcg.github.io/ChocolateOnJames/**
