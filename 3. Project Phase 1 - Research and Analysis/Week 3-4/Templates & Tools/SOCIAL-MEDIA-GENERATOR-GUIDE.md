# Chocolate on James - Social Media Image Generator Guide

## Overview

This interactive tool allows you to quickly create professional social media images for Instagram, Facebook, and other platforms without needing design software like Canva or Photoshop.

## How to Use

### 1. Open the Tool
Simply double-click the `social-media-image-generator.html` file to open it in your web browser (Chrome, Safari, Firefox, etc.)

### 2. Choose Your Platform
Select from preset sizes:
- **Instagram Post** (1080 × 1080) - Square format
- **Instagram Story** (1080 × 1920) - Vertical story format
- **Facebook Ad** (1200 × 628) - Landscape ad format
- **Portrait Post** (1080 × 1350) - Vertical feed format

### 3. Use Quick Templates
Choose from pre-designed templates:
- **Product Showcase** - Highlight your artisan chocolates
- **Special Offer** - Promote sales and discounts
- **Inspirational Quote** - Share chocolate wisdom
- **Announcement** - New products and events

### 4. Customize Your Content

#### Text Content
- **Main Headline**: Your primary message (e.g., "Handcrafted Artisan Chocolates")
- **Subtitle**: Supporting text or description (e.g., "Made with Love in Hamilton")

#### Colors
- Choose **Background Color** and **Text Color**
- Use preset chocolate-themed colors:
  - Dark Chocolate (#2d1810)
  - Milk Chocolate (#8B4513)
  - Gold (#D4AF37)
  - Cream (#f5f5dc)
  - Deep Brown (#4a2c2a)

#### Font Sizes
- Adjust **Main Text** size (30-120px)
- Adjust **Subtitle** size (16-60px)

#### Logo/Image
- Upload your logo or product images
- Toggle logo visibility on/off

### 5. Export Your Design
Three options available:
- **Download Image**: Save as PNG file to your computer
- **Copy to Clipboard**: Paste directly into social media platforms
- **Reset**: Start over with default settings

## Tips for Best Results

1. **Keep Text Short**: 5-7 words for headlines work best
2. **High Contrast**: Use light text on dark backgrounds or vice versa
3. **Upload Quality Images**: Use high-resolution product photos
4. **Test Multiple Versions**: Try different color combinations
5. **Save Your Favorites**: Download and organize by campaign

## Frequently Asked Questions

**Q: Can I use this offline?**
A: Yes! Once you've opened it once, the file works without internet.

**Q: Will my uploaded logo be saved?**
A: Logos are only stored in your browser session. Re-upload each time you open the tool.

**Q: Can I edit the downloaded images?**
A: Yes, they're standard PNG files that work with any photo editor.

**Q: What if text doesn't fit?**
A: Use the font size sliders or reduce the number of words.

**Q: Can I create custom sizes?**
A: The current version supports standard social media sizes only.
