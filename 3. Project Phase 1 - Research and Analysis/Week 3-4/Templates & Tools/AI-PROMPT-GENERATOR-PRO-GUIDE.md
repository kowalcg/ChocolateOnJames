### Midjourney
**Best for:** Artistic, magazine-quality images

**Pros:**
- Stunning artistic quality
- Fast generation (60 seconds)
- Great for food photography
- Highly customizable with parameters

**Cons:**
- Discord-based (learning curve)
- Requires paid subscription
- Less precise with text elements
- Can be unpredictable

**When to use:** Hero images, artistic content, magazine-quality shots, social media features

**Parameters to know:**
- `--ar 1:1` = Square (Instagram posts)
- `--ar 9:16` = Vertical (Stories)
- `--ar 2:3` = Pinterest tall pin
- `--v 6` = Version 6 (latest)
- `--style raw` = Photorealistic
- `--q 2` = Highest quality

## üîç Troubleshooting Common Issues

### Issue: Colors Don't Match Brand
**Solution:**
- Emphasize colors in prompt: "Ensure rich chocolate brown #3E2723"
- Regenerate 5-10 variations
- Adjust in post-processing
- Try different AI platform (Gemini often better with colors)

### Issue: Too Much Text/Clutter
**Solution:**
- Add to prompt: "Minimal or no text, clean composition"
- Use negative prompt: "text, words, letters" (Stable Diffusion, Leonardo)
- Emphasize "negative space" and "clean aesthetic"

### Issue: Doesn't Look Professional
**Solution:**
- Add quality modifiers: "professional commercial photography, magazine-quality, high-end"
- Try Midjourney with `--style raw --q 2`
- Reference specific photography style: "professional food photography for Bon App√©tit magazine"

### Issue: Wrong Composition
**Solution:**
- Be very specific: "centered composition with product in middle third"
- Specify camera angle: "45-degree overhead angle"
- Mention "rule of thirds" or "negative space on left"

### Issue: Chocolate Looks Artificial
**Solution:**
- Add: "photorealistic, authentic chocolate texture, real chocolate"
- Use Midjourney with `--style raw`
- Emphasize: "professional food photography, natural lighting, real product"

### Issue: Can't Get Consistent Style
**Solution:**
- Use Leonardo AI with same settings/model
- Save successful prompts and reuse with small variations
- Batch generate with same prompt
- Use style reference images (Midjourney feature)

## ‚ùì Frequently Asked Questions

**Q: Which AI platform should I start with?**
A: Start with Google Gemini (it's free!) or ChatGPT if you have Plus. Both are beginner-friendly and produce good results.

**Q: How many images should I generate per prompt?**
A: Generate 5-10 variations per prompt, then select the best 1-2. This gives you options while managing costs.

**Q: Can I edit AI-generated images?**
A: Yes! Edit in Canva, Photoshop, or any image editor. Add text, adjust colors, crop, etc.

**Q: Are AI images safe for commercial use?**
A: Depends on platform. ChatGPT/DALL-E, Gemini, and Adobe Firefly are commercially safe. Midjourney requires paid subscription. Check platform terms.

**Q: What if the image doesn't match brand colors exactly?**
A: Minor color adjustment in Canva or Photoshop is normal. Mention colors explicitly in prompt for better accuracy.

**Q: How long does image generation take?**
A: 30 seconds to 2 minutes per image, depending on platform and settings.

**Q: Can I use the same prompt multiple times?**
A: Yes! Same prompt generates different variations each time.

**Q: What's the difference between platforms?**
A: Each has strengths: ChatGPT (easy), Midjourney (artistic), Gemini (free), Ideogram (text), Firefly (commercial safe).

**Q: Do I need all the AI platforms?**
A: No. Start with 1-2. Add more as you learn what works for your needs.

**Q: How do I maintain brand consistency?**
A: Use same base prompts, similar settings, consistent style selection, and save successful prompts.
