## üìä HOW THE TOOL WORKS

### **User Workflow:**
1. Select AI platform (Gemini, ChatGPT, etc.)
2. Choose content category (8 options + custom)
3. Select platform/format (Instagram Post, Story, Facebook, Pinterest)
4. Choose season/occasion (Jan-Dec + custom)
5. Select target audience (7 options + custom)
6. Pick image style (Photography, Lifestyle, Minimalist, Artistic)
7. Check image elements (text space, logo, location, people)
8. Check caption options (emojis, questions, CTAs, GIFs, formal)
9. Add product name and special notes
10. Click "Generate Complete Content"

### **Output:**
- 1 AI image prompt (structured, professional-grade)
- 10 unique social media captions with hashtags
- Platform-optimized formatting
- Export and copy functions

### **Regenerate Feature:**
- Shuffles the 10 captions into random order
- Can change parameters before regenerating
- Creates fresh variations
