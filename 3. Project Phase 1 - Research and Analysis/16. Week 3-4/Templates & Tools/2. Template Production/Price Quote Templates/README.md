## Price Quote Templates – Agent Brief

- **Purpose:** create standardized quote formats for retail inquiries, corporate gifting, and wholesale orders.
- **Deliverables:** editable Google Doc template, Google Sheets calculator for volume tiers, and PDF export. Include sections for scope summary, product breakdown, customization fees, fulfillment timeline, payment terms, and signature block.
- **Branding:** match letterhead styling (logo top-left, contact info footer, warm neutral colors). Ensure consistent typography and clear table formatting.
- **Workflow:** draft base content → build dynamic fields for quantity, unit price, rush fees → test formulas for tiered pricing → provide instructions for duplicating and sending to clients.
- **Handoff:** supply usage checklist, version control tips, and a short FAQ on adjusting tax/shipping fields.
