## Analytics Dashboard Templates – Agent Brief

- **Goal:** supply dashboards that consolidate website, email, and sales metrics. Focus on Pavement analytics exports, Google Analytics 4, and social performance.
- **Deliverables:** Google Data Studio/Looker Studio template (shareable link), Google Sheets tracker with import ranges, and instructions for connecting Pavement data or manual CSV uploads.
- **Key metrics:** traffic sources, product views, add-to-cart, conversion rate, average order value, top SKUs, corporate inquiry pipeline, email campaign performance, UGC campaign results.
- **Process:** map data sources → define visual layout (scorecard + trend + breakdown) → build filter controls (date range, channel, product type) → test with sample data.
- **Handoff:** include connection guide, data refresh schedule, access permission tips, and glossary of metrics.
