### How to work with multiple agents

1. Assign each agent one or two folders inside `2. Template Production`.
2. Agents follow the README brief inside their folder, save drafts in the same location, and update status here once approved.
3. Maintain a simple naming convention: `COJ-[TemplateName]-v1.ext`.
4. When deliverables are final, export to PDF (or HTML for emails) and add a short usage note.
