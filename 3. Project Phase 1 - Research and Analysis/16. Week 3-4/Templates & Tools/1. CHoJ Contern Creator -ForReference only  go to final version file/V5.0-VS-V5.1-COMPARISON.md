## Migration Guide

### From v5.0 to v5.1

#### What You Need to Do
**Nothing!** Just replace the HTML file.

#### Will My Workflows Break?
**No!** Everything works exactly the same.

#### Do I Have to Use the Image Feature?
**No!** It's completely optional.

#### Can I Still Use Presets?
**Yes!** All 4 presets work identically.

#### What About My Bookmarks?
Replace file path in bookmark, that's all.

#### Any Data to Migrate?
**No!** Tool doesn't store data (by design).

## When to Use Which Version

### Use v5.0 If:
- You don't need image analysis
- You prefer minimal interface
- You have slow internet (smaller file)
- You never work with reference images
- You're satisfied with current workflow

### Use v5.1 If:
- You want to match existing styles
- You work with inspiration images
- You need color palette extraction
- You want 4 prompt variations per image
- You want the fastest workflow possible
- You create content regularly
