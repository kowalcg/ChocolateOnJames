## Pro Tips

### Image Upload Tips
- **Best images:** Clear product photos, professional lighting
- **Avoid:** Blurry photos, heavy filters, busy backgrounds
- **Format:** JPG or PNG work best
- **Size:** Under 5MB recommended

### Variation Selection Guide
- **Exact Match:** When you found the perfect style
- **Inspired:** When you want variety but same quality
- **Premium:** For hero images, ads, key campaigns
- **CHoJ Branded:** For all social media and daily posts

### Platform Optimization
- **Instagram Post:** Square format, vibrant, eye-catching
- **IG Story:** Vertical format, text overlay, urgent feel
- **Facebook Ad:** Clear CTA, benefit-focused, conversion-oriented
- **Pinterest:** Tall format, inspirational, lifestyle context

### Social Caption Tips
- **Enable emojis:** More engaging on Instagram/Facebook
- **Enable questions:** Boosts engagement and comments
- **Strong CTA:** Better for promotional content
- **Formal tone:** Better for corporate/B2B gifting

## Troubleshooting

### "Image won't upload"
- Check file format (JPG, PNG, WebP only)
- Try different upload method (click vs paste)
- Refresh page and try again

### "Analysis not showing"
- Wait 1-2 seconds for processing
- Check if image preview appeared
- Current version uses demo data (always works)

### "Use button not working"
- Scroll down to ensure output area is loaded
- Try copying prompt manually instead
- Refresh page if issue persists

### "No captions generating"
- Ensure platform/category is selected
- Click "Generate All Content" button
- Check that JavaScript is enabled in browser
