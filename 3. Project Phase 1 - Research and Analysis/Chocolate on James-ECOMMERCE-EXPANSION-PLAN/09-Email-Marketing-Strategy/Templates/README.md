## Email Marketing Templates – Agent Brief

- **Goal:** deliver 10–15 modular email templates covering promos, storytelling, product launches, corporate gifting, and reminders. Must reflect Chocolat on James tone (warm, premium, neighbourhood-focused).
- **Format:** build master template in Canva or Google Docs for layout reference, then translate into editable HTML blocks (header, hero, body, CTA, footer). Provide both `.html` and `.pdf` exports, plus a short implementation note for common ESPs (Klaviyo/Mailchimp).
- **Assets to include:** logo lockups, color palette, typography callouts, placeholder product shots, seasonal sections (Holiday, Valentine’s, Corporate Gifts), testimonial block, FAQ/Shipping details.
- **Workflow:** draft copy snippets → design base layout → create variants (newsletter, promo, nurture, re-engagement) → package into folder.
- **Handoff checklist:** usage instructions, editable source file link, image folder, QA via Litmus/Email on Acid (or document manual tests).
